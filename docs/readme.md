# BPP — Bibliografia Publikacji Pracowników

Bibliografia Publikacji Pracowników to system informatyczny do zarządzania
bibliografią publikacji pracowników naukowych. Oprogramowanie przeznaczone
jest dla bibliotek naukowych i uniwersyteckich w Polsce.

Oprogramowanie dystrybuowane jest na zasadach otwartoźródłowej [licencji MIT](https://pl.wikipedia.org/wiki/Licencja_MIT).

Pełne informacje o projekcie znajdują się w pliku [README.md](https://github.com/iplweb/bpp/blob/dev/README.md) w głównym
katalogu repozytorium.
