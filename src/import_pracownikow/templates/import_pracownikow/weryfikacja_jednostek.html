    Weryfikacja jednostek

    Jednostki wymienione w pliku, których nie dopasowano dokładnie do bazy. Wskaż, co zrobić z każdą, zanim zapiszesz import. Domyślnie każda zostanie utworzona (lub użyte zostanie auto-dopasowanie) — możesz to zmienić.

    {% if not decyzje_brak and not decyzje_zgadywanie %}
    {# brak decyzji = wszystkie jednostki dopasowane dokładnie #} Wszystkie jednostki z pliku zostały dopasowane — nie ma nic do weryfikacji.
    Wróć do przeglądu {% else %}
    {% csrf_token %} {% if decyzje_zgadywanie %}

    {% endif %} {% if decyzje_brak %}
