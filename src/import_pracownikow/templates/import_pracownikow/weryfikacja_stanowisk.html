    Weryfikacja stanowisk dydaktycznych

    Stanowiska dydaktyczne wymienione w pliku, których nie dopasowano dokładnie do bazy. Wskaż, co zrobić z każdym, zanim zapiszesz import. Domyślnie każde zostanie utworzone (lub użyte zostanie auto-dopasowanie) — możesz to zmienić.

    {% if not decyzje_brak and not decyzje_zgadywanie %}
    {# brak decyzji = wszystkie stanowiska dopasowane dokładnie #} Wszystkie stanowiska z pliku zostały dopasowane — nie ma nic do weryfikacji.
    Wróć do przeglądu {% else %}
    {% csrf_token %} {% if decyzje_zgadywanie %}

    Dopasowane automatycznie

    Nazwa z pliku jest bardzo zbliżona do istniejącego stanowiska — zweryfikuj automatyczny wybór.

    {% for dec in decyzje_zgadywanie %} {% endfor %}
    Nazwa z pliku Osób Auto-stanowisko Decyzja Mapuj na
    {{ dec.nazwa_zrodlowa }} {{ dec.liczba_osob }} {% if dec.auto_stanowisko %} {{ dec.auto_stanowisko.skrot }} {% if dec.auto_similarity %} {{ dec.auto_similarity|floatformat:2 }} {% endif %} {% else %}—{% endif %}
    {% endif %} {% if decyzje_brak %}

    Do utworzenia

    Brak dopasowania w bazie — domyślnie zostaną utworzone jako nowe stanowiska. Możesz poprawić nazwę i skrót przed zapisem.

    {% for dec in decyzje_brak %} {% endfor %}
    Nazwa z pliku Osób Decyzja Nazwa Skrót Mapuj na
    {{ dec.nazwa_zrodlowa }} {{ dec.liczba_osob }}
    {% endif %}
