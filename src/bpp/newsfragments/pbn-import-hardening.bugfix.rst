Import z PBN jest teraz odporny na niekompletne i zdryfowane rekordy — pojedynczy zły wpis nie wywala już całego wsadu, tylko jest pomijany, liczony i raportowany. Rekordy strukturalnie nieimportowalne (bez wersji bieżącej, bez typu obiektu, rozdział bez książki nadrzędnej) są jawnie pomijane; nieznane klucze z PBN (np. ``reviewers``) trafiają do adnotacji zamiast przerywać import; zdublowane lub brakujące rekordy słownika języków degradują do języka domyślnego zamiast wywalać rekord; a książki bez wydawcy poprawnie wchodzą (i nie wywracają post-importowej punktacji zwartych).
