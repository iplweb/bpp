Import pracowników: gdy dopasowanie po nazwisku trafi na rekord-duplikat, operacje zatrudnienia (przypisanie do jednostki, zmiana miejsca pracy, przepięcia) kierowane są teraz do rekordu głównego autora — oryginału z ORCID, tytułem i odpowiednikiem w API instytucjonalnym PBN — bez scalania rekordów. Widoki rezultatów, odpięć i audytu wzbogacono o ORCID, linki do profilu autora i panelu admina oraz wskaźnik pochodzenia z API instytucjonalnego PBN; dodano zbiorcze zaznaczanie odpięć z bieżącego filtra tabeli oraz aktywne wyszukiwanie i stronicowanie (DataTables) na ekranie audytu. Przekolorowano plakietki statusów dopasowania autora (m.in. „wielu kandydatów" na czerwono, osobny kolor dla wyboru operatora).
